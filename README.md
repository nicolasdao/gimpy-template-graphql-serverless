# Gimpy Template - GraphQL Web Server For Google Cloud Functions or Firebase Functions

## Intro
Use [gimpy](https://github.com/nicolasdao/gimpy) to install and deploy this template. To install gimp, simply execute the following:
```
npm install gimpy -g
```

## How To Use It
```
gimp new graphql-serverless your-app
cd your-app
npm install
gimp deploy
```
